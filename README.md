# 🏋️ Body Sport Academia — Website

Projeto de website profissional desenvolvido para a academia Body Sport, localizada em Castelo - ES.

## 🚀 Objetivo

Criar uma presença digital moderna, otimizada para conversão de alunos e autoridade online.

## ✨ Funcionalidades

* Landing page moderna e responsiva
* Integração com WhatsApp
* Seção de planos e serviços
* Depoimentos de alunos
* Design focado em conversão

## 🛠️ Tecnologias utilizadas

* React
* Vite
* TypeScript
* TailwindCSS
